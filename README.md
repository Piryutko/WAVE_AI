# WAVE AI Landing Page

[English](#english) | [Русский](#русский) | [日本語](#日本語)

## English

### Overview
WAVE AI is a premium, high-tech landing page designed to showcase a suite of AI-powered modules. The interface follows a "Wave" design language, characterized by minimalism, terminal-style typography, and smooth interactive elements.

### Key Features
- **Draggable Carousel**: Interactive module selection with a glassmorphism aesthetic.
- **Dynamic Interaction**: The main page title changes color and adds a glow effect based on the selected module (Dune for TaskWave, Blue for TimeWave, Green for ToneWave).
- **Japanese Marquee**: A scrolling terminal-style banner at the top of the page.
- **Micro-animations**: Subtle hover effects and smooth transitions for a premium feel.

### Tech Stack
- [Svelte 5](https://svelte.dev/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/) (for utility layout)
- [Lucide Svelte](https://lucide.dev/) (Icons)

## Русский

### Обзор
WAVE AI — это премиальный высокотехнологичный лендинг, разработанный для демонстрации набора модулей на базе искусственного интеллекта. Интерфейс следует дизайн-коду "Wave", отличающемуся минимализмом, типографикой в стиле терминала и плавными интерактивными элементами.

### Основные характеристики
- **Интерактивная карусель**: Выбор модулей с эстетикой матового стекла (glassmorphism).
- **Динамическое взаимодействие**: Заголовок страницы меняет цвет и добавляет эффект свечения в зависимости от выбранного модуля.
- **Японская «бегущая строка»**: Скроллящийся баннер в стиле терминала в верхней части страницы.
- **Микро-анимации**: Тонкие эффекты при наведении и плавные переходы для премиального вида.

## 日本語

### 概要
WAVE AIは、AI搭載モジュールのスイートを紹介するために設計された、プレミアムでハイテクなランディングページです。インターフェースは「Wave」デザイン言語に従っており、ミニマリズム、ターミナルスタイルのタイポグラフィ、スムーズなインタラクティブ要素が特徴です。

### 主な機能
- **ドラッグ可能なカルーセル**: グラスモーフィズムな美学を備えたインタラクティブなモジュール選択。
- **動的なインタラクション**: メインページのタイトルは、選択したモジュールに基づいて色が変わります。
- **日本語の「電光掲示板」**: ページ上部にある、ターミナルスタイルのスクロールバナー。
- **マイクロアニメーション**: プレミアムな感覚のための、微妙なホバー効果とスムーズなトランジション。
